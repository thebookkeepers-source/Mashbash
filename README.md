# Mashbash

Mashbash is a premium food delivery web app prototype for customers, riders, and restaurants.

## What is included

- Responsive customer home page
- Food categories and featured restaurants
- Cart and checkout summary
- Rider login panel
- Live order tracking style section
- Premium UI using plain HTML, CSS, and JavaScript

## How to run

Open `index.html` in a browser.

No build step is required.
